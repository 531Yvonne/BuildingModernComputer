# This source code contains the compiler to compile .jack file input or a directory contains .jack file(s) to .vm files under the same directory as the input.


* This is a python3 program, no need to compile.

* To run the code: run the line below in your command line in the src directory:

python3 jack_compiler.py <replace by the path to input file or directory>

The corresponding .vm output files will be generated to the same directory.
