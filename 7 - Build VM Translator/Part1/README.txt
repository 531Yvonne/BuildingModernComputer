# This source code contains a program to convert .vm input to assembly code at a .asm file to the same directory as the input file.


* This is a python3 program, no need to compile.


* To run the code: run the line below in your command line in the src directory:


python3 project7.py <replace by the path of your input file (including filename.vm)>


the corresponding .asm output file will be generated to the same directory


* Functionality in detail:

<works> ---- Arithmetic Command
<works> ---- Basic Memory Access Command: push pop constant/local/argument/this/that/temp
<works> ---- Memory Access Command with pointer 
<works> ---- Memory Access Command with static
